# Clipstream


## Introduction

A very simple tool for caching clips in your scripts from stdin.

```
cat /path/to/file.txt | pushclip push
```


## Deployment

### Installation

Clone the repository and run:

```
sudo make install
```

### Configuration

**Nginx**: Refer to the `/conf/clipstream.nginx.conf` file.
